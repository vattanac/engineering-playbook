# Kubernetes Stateless vs Stateful — Complete Beginner Guide

A comprehensive, interactive HTML guide that explains Kubernetes stateless and stateful concepts from zero to confident — built for DevOps beginners.

## Preview

The guide features a modern dark-themed UI with a fixed left sidebar navigation, animated diagrams, syntax-highlighted YAML examples, and scroll-tracking progress.

## What's Inside

The guide covers 17 sections organized into 5 groups:

**Foundations** — What is "State", real-world analogies (restaurant vs hospital), deep dives into stateless and stateful apps, and a side-by-side comparison table.

**Kubernetes Basics** — Core concepts (Pod, Deployment, StatefulSet, Service, PV/PVC), step-by-step deployment of stateless apps, and step-by-step deployment of stateful apps.

**Advanced Topics** — Why stateful is harder (the 5 challenges), persistent volumes deep dive, and headless services explained.

**Full Examples** — Production-ready YAML for a stateless Nginx web server and a stateful MySQL database, plus how scaling works differently.

**Reference** — When to use which, common beginner mistakes, and a cheat sheet with kubectl commands and a decision flowchart.

## Features

- Dark theme with animated background grid
- Fixed left sidebar with grouped navigation and scroll-spy highlighting
- Animated visual diagrams (load balancer flow, stateful architecture, PV/PVC chain, decision flowchart)
- Syntax-highlighted YAML code blocks with terminal-style dots
- Animated progress bars showing stateless vs stateful complexity
- Color-coded comparison tables
- Interactive hover effects on diagram nodes
- Mobile responsive with hamburger menu toggle
- Reading progress indicator in sidebar footer

## How to Use

Open `kubernetes-stateless-vs-stateful-guide.html` in any modern web browser. No build tools, dependencies, or server required — it's a single self-contained HTML file.

## Tech Stack

HTML5, CSS3 (custom properties, grid, flexbox, animations, glassmorphism, backdrop-filter), vanilla JavaScript (IntersectionObserver, scroll events), Google Fonts (Inter, JetBrains Mono).
